If you already have the Zoom account

  • 1. Open Zoom desktop client and sign in.
  • 2. Click your profile picture then click Check for Updates

    Image source: https://support.zoom.us/hc/en-us/articles/201362233-Where-Do-I-Download-The-Latest-Version-
  • 3. If there is a newer version, Zoom will download and install it.

If you don't have zoom account

  • 1. Open Zoom desktop client
  • 2. At the bottom of the application you see current version

    E.g.: Version: 5.0.403652.0509
  • 3. When first digit is below 5
    • Click on the following link to download the lastest version
      Zoom desktop client
      Click on downloaded file and install new app version.
